# Lesson 1: Big O Notation

## Learning objectives

By the end of this lesson, I should be able to:

- Explain what Big O notation means.
- Identify common time complexities.
- Compare the efficiency of algorithms.
- Analyze loops and nested loops.
- Explain why Big O matters in AI and machine learning.
- Distinguish between time complexity and space complexity.

## Topics

1. Input size and operations
2. Constant time — O(1)
3. Linear time — O(n)
4. Quadratic time — O(n²)
5. Logarithmic time — O(log n)
6. Linearithmic time — O(n log n)
7. Simplifying Big O expressions
8. Best, average, and worst cases
9. Time complexity
10. Space complexity
11. AI and machine learning applications

## AI and ML connection

Big O helps AI and ML engineers understand whether an algorithm can handle:

- Large datasets
- High-dimensional features
- Model training operations
- Search and recommendation systems
- Real-time predictions
- Large numbers of users or requests
